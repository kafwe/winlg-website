Abstract Wikipedia Toolkit

Developer(s) University of Cape Town
Initial release 2024
Platform Web-based
Type Content generation tools
License Open-source
Website abstractwikipedia.org

The Abstract Wikipedia Toolkit is a collection of software tools designed to facilitate the creation and management of content for Abstract Wikipedia. Developed by researchers at the University of Cape Town, the toolkit aims to streamline the process of contributing to Abstract Wikipedia by providing intuitive interfaces for content generation and management.

Introduction

Abstract Wikipedia is an ambitious project that seeks to create a language-independent version of Wikipedia. The Abstract Wikipedia Toolkit was developed to address the challenges of creating and managing content for this multilingual, abstract representation of knowledge.

Components

The Abstract Wikipedia Toolkit consists of two main components: CRAFT and TempTing. These tools work together to provide a comprehensive solution for content creation and management in Abstract Wikipedia.

CRAFT

CRAFT (Constructor and Retrieval API for Template Filling) is an HTTP API designed for multilingual content selection. It processes constructors and converts them into SPARQL queries to retrieve content from Wikidata. This component improves the discoverability of relevant content and facilitates the integration of Wikidata information into Abstract Wikipedia articles.

TempTing

TempTing (Template Tinkering) is a web-based application that aids in the creation and management of templates for Abstract Wikipedia. It provides an integrated development environment for template editing, complete with features such as syntax highlighting, auto-completion, and error detection. TempTing simplifies the process of creating and modifying templates, which are crucial for generating natural language content from abstract representations.

Project Goals

The primary goals of the Abstract Wikipedia Toolkit are:

  • To simplify the process of contributing to Abstract Wikipedia
  • To improve the quality and consistency of generated content
  • To facilitate multilingual content creation and management
  • To enhance the integration between Wikidata and Abstract Wikipedia

Usage

The Abstract Wikipedia Toolkit is designed for use by Wikipedia editors, content creators, and developers working on the Abstract Wikipedia project. Users can access CRAFT through its API endpoints, while TempTing is available as a web application. Detailed documentation and usage instructions are provided for both components on the project's official website.

Future Development

The development team at the University of Cape Town continues to work on improving and expanding the Abstract Wikipedia Toolkit. Future plans include:

  • Enhanced natural language generation capabilities
  • Improved integration with existing Wikipedia editing tools
  • Support for a wider range of languages and linguistic structures
  • Development of additional tools to further streamline the content creation process
